# Installation Guide

## Prerequisites
- Modern browser (Chrome, Firefox, Edge, Safari)
- Optional: Python 3 for local static hosting

## Steps
1. Clone repository:
   ```bash
   git clone https://github.com/arief121440022/arief-gis-portfolio.git
   cd arief-gis-portfolio
   ```
2. Start local server (recommended):
   ```bash
   python3 -m http.server 8000
   ```
3. Access website at `http://localhost:8000`.

## Quick verification checklist
- Navigation links scroll to correct section
- Mobile menu opens and closes
- Contact form validates inputs
- Scroll-to-top button appears after scrolling
