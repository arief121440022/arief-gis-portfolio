## 🚀 Installation & Setup

### Prerequisites
- Web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code recommended)
- Git (optional)

### Method 1: Local Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/arief121440022/arief-gis-portfolio.git
   cd arief-gis-portfolio
   ```

2. **Open in browser**
   - Double-click `index.html`, or
   - Use a local server:
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Python 2
     python -m SimpleHTTPServer 8000
     
     # Node.js
     npx http-server
     ```
   - Visit: `http://localhost:8000`

### Method 2: Direct Download
1. Download the repository as ZIP
2. Extract the files
3. Open `index.html` in your browser

## 💡 Usage

### Customization

#### 1. Update Personal Information
Edit `index.html` and update:
- Name in navbar and hero section
- Contact information (email, phone)
- Social media links
- Professional description

#### 2. Add Your Photo
Replace the map icon in hero section with your photo

#### 3. Update Content Sections
- Edit experience items in the tabbed section
- Update skill tags and progress bars
- Modify portfolio project cards
- Change colors by editing CSS variables in `css/style.css`

#### 4. Download CV Button
Place your CV file in `assets/Arief_Cv.pdf`
