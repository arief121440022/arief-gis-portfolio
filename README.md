# Arief GIS Portfolio

Production-ready personal portfolio for **Arief Ramadhan** (GIS Analyst & Geospatial Specialist).

## Features
- Semantic HTML5, responsive CSS, and modern vanilla JavaScript
- Smooth scrolling, mobile menu, intersection observer animations
- Contact form validation with accessible feedback
- Scroll-to-top button and keyboard-focus states
- SEO files (`robots.txt`, `sitemap.xml`) and favicon

## Structure
- `index.html` — Single-page portfolio
- `css/style.css` — Main styling
- `css/responsive.css` — Mobile/tablet breakpoints
- `js/script.js` — Navigation, animation, scrolling
- `js/form-handler.js` — Form validation behavior
- `assets/resume.pdf` — Downloadable resume file

## Local usage
Open `index.html` directly, or run a static server:

```bash
python3 -m http.server 8000
```

Then open: `http://localhost:8000`
