# Deployment Guide

## GitHub Pages (recommended)
1. Push code to default branch.
2. Open **Repository Settings → Pages**.
3. Source: **Deploy from branch**.
4. Branch: `main` (or target branch), folder `/root`.
5. Save and wait for deployment.

Published URL pattern:
`https://arief121440022.github.io/arief-gis-portfolio/`

## Other static hosting options
- Netlify (drag-and-drop or Git integration)
- Vercel (import repository as static site)
- Shared hosting (upload all files to web root)
