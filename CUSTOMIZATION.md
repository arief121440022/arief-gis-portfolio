# Customization Guide

## Update profile content
- Edit text sections in `index.html`:
  - Hero, About, Skills, Projects, Timeline, Contact

## Update theme
- Edit CSS variables in `css/style.css` under `:root`.

## Add new projects
- Duplicate a `.project-card` block inside `#projects`.

## Update form behavior
- Modify validation rules in `js/form-handler.js`.

## Replace resume
- Replace `assets/resume.pdf` with final resume file (same filename).
